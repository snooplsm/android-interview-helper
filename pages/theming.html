Light & Dark Theme

📖 Docs

Define two ColorScheme objects with lightColorScheme() and darkColorScheme(), then select based on isSystemInDarkTheme().

Kotlin
📚 Deep Dive

Dynamic Color (Android 12+): Use dynamicLightColorScheme(context) and dynamicDarkColorScheme(context) to pull colors from the user's wallpaper.

Color roles: MD3 has 29 color roles. Key ones: primary, onPrimary, primaryContainer, onPrimaryContainer, surface, onSurface, background, error.

Access colors anywhere: MaterialTheme.colorScheme.primary

Interview tip: Explain the difference between surface and background — surfaces layer on top of background, used for cards, sheets, etc.

CompositionLocal

📖 Docs

Implicit dependency injection through the composition tree. Avoids prop-drilling. Use staticCompositionLocalOf for rarely-changing values and compositionLocalOf for frequently-changing ones.

Kotlin
📚 Deep Dive

staticCompositionLocalOf: When the value changes, the entire subtree recomposes. Use for stable objects (colors, typography).

compositionLocalOf: Only composables that read the value recompose when it changes. Use for frequently-changing values (theme mode, locale).

Built-in locals: LocalContext, LocalDensity, LocalConfiguration, LocalLifecycleOwner, LocalFocusManager.
